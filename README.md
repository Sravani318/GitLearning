Basic git commands:
1. git -v --> used to check the version of the git
2. git init --> initializes a empty local repository in the folder.
3. before working on the project we need to configure the git
4. i. git config --global user.name "name of the user"
5. ii. git config --global user.email "email address of the user"
6. git status --> gives the status of current repository
7. git add filenam --> adds the file to the staging area
8. git commit -m "write message related to commit"
9. git log --> lists out the commit so far we have done in this repository.
10. git add . -->adds all the files in that folder to staging area.
11. git show 1st 8 characters from the commit id --> shoes the details of the particular id.
12. git ls-files --> list out the files in that folder
13. HOW TO PUSH FILES FROM LOCAL REPOSTIORY TO REMOTE REPOSITORY:
1. git remote add origin link of the repo  --> used to link remote repository to the local repository
2. git remote -v --> checks the remote repository
3. git push origin master --> pushes the code from local repository to remote repository.
